### 正则表达式，替换路径路径

因为 vue 解析相对路径，所以要以 `./` 开头路径才可以被 vue 替换

```
!\[(.*?)\]\((?!\.\/)(.*?\.assets/.*\.(png|jpg))\)
```

```
![$1](./$2)
```
