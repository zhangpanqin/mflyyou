# SpringBoot 日常技术栈

- gradle
- [mockserver](https://github.com/mock-server/mockserver) 启动一个 server ，可以帮助我们去 test 请求。比如 feign client

- [embedded-redis](https://github.com/kstyrc/embedded-redis)
